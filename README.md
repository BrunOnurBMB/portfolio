> 💡 *Você pode acessar o site em produção aqui:*  
> 🔗 [https://seu-link-na-vercel.vercel.app](https://portifolio-mu-navy-67.vercel.app/)

## 🚀 Tecnologias Utilizadas

- **HTML5**
- **Tailwind CSS**
- **JavaScript**
- **ThreeJS** (Efeito com particulas)
- **Vercel** (Deploy)
  

---

## 📁 Estrutura do Projeto

```
├── assets/ │
      ├── imgs/ # Imagens │
```
